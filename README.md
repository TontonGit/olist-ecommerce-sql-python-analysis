# 📊 Olist E-commerce Data Analysis ( SQL & Python)

## 📌 Project Overview

This project analyzes the **Olist Brazilian E-commerce dataset** to uncover key business insights related to:
* Revenue distribution
* Customer behavior
* Business growth trends
* Customer segmentation
The analysis combines **SQL for data extraction and Python for data analysis and visualization**, demonstrating an end-to-end
analytic s workflow.

# 🗂️ Dataset

The dataset contains transactional data from a Brazilian e-commerce marketplace, including: 
* Customers
* Orders
* Orders items
* Geographic information

# 🎯 Business Questions

This analysis focuses on answering the following questions:
1. **Where does revenue come from**?
2. **How has the business grown over time**?
3. **Who are the customers**
4. **Do customers return and make repeat purchases**?
